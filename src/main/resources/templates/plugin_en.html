Plugin

Features

  • If an error occurs on the web page you are viewing, it will automatically be sent to the error repository server
    The sent error information is temporarily saved in the browser
  • By clicking the icon, you can check the list of information sent from your own browser
    By clicking the [Request Delete] button in the list, you can delete the data saved on the server

Items to Collect*1

  • URL
  • Date and time
  • Browser environment
  • Error message
  • Stack trace
  • File name
  • Line number, Column number
  • Only when approved*2 Query part of URL (part after "?")
  • Only when approved*2 Cookie
*1 The Item was selected based on whether it helps to reproduce the error occurrence situation
*2 See below

About some items

  • Query part of URL/Cookie are not disclosed because they may contain personal information
    It does not collect them by initial setting
  • We will provide information only to page developers for collecting and reproducing error situation only when you give permission from setting screen
